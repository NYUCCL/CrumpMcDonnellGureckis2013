Instructions

To learn the labels of the items, you will see them laid out in a grid like this:

You can click any square to see its label. Try it!

This is just an example using X and Y as category labels. You will have to learn about categories A and B in the actual task.

On each round of training, you will have 16 chances, which will be counted down in a diagram like the one below. Each red mark symbolizes a chance you have to see an item's label. Each time you click on a square, one of these red marks will turn grey.

When you run out of examples, the experiment will proceed to a test phase where you will be asked to identify each of the items in turn.

When you feel you understand how to query for a category label and the meaning of the timer diagram, press Next to continue.
